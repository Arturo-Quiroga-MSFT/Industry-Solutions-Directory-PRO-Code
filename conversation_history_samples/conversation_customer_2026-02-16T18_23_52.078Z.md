**SQL**:
```sql
SELECT DISTINCT TOP 50 solutionName, orgName, industryName, solutionAreaName, marketPlaceLink, solutionOrgWebsite, geoName, solutionPlayName, solutionDescription
FROM dbo.vw_ISDSolution_All
WHERE solutionStatus = 'Approved'
  AND industryName = 'Education'
  AND (
    solutionName LIKE '%campus management%'
    OR solutionName LIKE '%administration%'
    OR solutionDescription LIKE '%campus management%'
    OR solutionDescription LIKE '%administration%'
    OR solutionName LIKE '%administrative%'
    OR solutionDescription LIKE '%administrative%'
  )
```

**Explanation**: This query lists approved solutions for the Education industry that are related to campus management or administrative functions. It searches both the solution name and description for keywords related to 'campus management' and 'administration'. Results include core details important for Microsoft sellers, with the solution description at the end for readability.
