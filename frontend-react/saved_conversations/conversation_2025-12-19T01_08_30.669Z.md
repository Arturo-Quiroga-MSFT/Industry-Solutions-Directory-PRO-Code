## Query 2

**Question**: Show me all solutions from Striim

**Result**: Found 31 results

**SQL**:
```sql
SELECT DISTINCT 
    solutionName,
    orgName,
    industryName,
    solutionAreaName,
    marketPlaceLink,
    solutionDescription
FROM dbo.vw_ISDSolution_All
WHERE orgName = 'Striim'
  AND solutionStatus = 'Approved';
```

**Explanation**: This query returns all approved ISD solutions published by the partner Striim. It uses DISTINCT to avoid duplicates from the denormalized view and includes the required core fields, with solutionDescription as the last column.
