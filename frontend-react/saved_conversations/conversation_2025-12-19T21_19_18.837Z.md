**SQL**:
```sql
SELECT DISTINCT TOP 50 
    solutionName,
    orgName,
    industryName,
    solutionAreaName,
    theme,
    subIndustryName,
    solutionDescription
FROM dbo.vw_ISDSolution_All
WHERE solutionStatus = 'Approved'
  AND (
        solutionName LIKE '%regulatory compliance%'
        OR solutionDescription LIKE '%regulatory compliance%'
        OR theme LIKE '%regulatory compliance%'
        OR subIndustryName LIKE '%regulatory compliance%'
      )
ORDER BY solutionName;
```

**Explanation**: Because the phrase 'regulatory compliance' forms a clear 2-word semantic concept, the query searches for the full phrase across relevant descriptive fields. The query returns approved solutions whose names or descriptions explicitly mention regulatory compliance.
